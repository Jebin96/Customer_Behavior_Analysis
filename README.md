📊 Customer Shopping Data Analytics Project
1️⃣ Overview

This project demonstrates a complete end-to-end Data Analytics workflow, transforming raw customer shopping data into actionable business insights.

The project covers:

Data loading and preprocessing using Python

Exploratory Data Analysis (EDA)

Data cleaning and transformation

SQL analysis across multiple databases (PostgreSQL, Oracle, MySQL, SQL Server)

Interactive Power BI dashboard development

Analytical report creation

Executive presentation built using Gamma
